CSS Outline Shorthand

Properti ini outline adalah properti singkatan untuk menetapkan properti kerangka individual berikut :

  • outline-width
  • outline-style ( diperlukan )
  • outline-color
Properti outline ditentukan sebagai satu, dua, atau tiga nilai dari daftar di atas. Urutan nilai tidak menjadi masalah.
Contoh berikut menunjukkan beberapa kerangka yang ditentukan dengan outline properti shorthand :

Properti outline shorthand

Dotted red outline

5px solid yellow

CSS Outline Width

Properti outline-width menentukan width outline, dan dapat memiliki salah satu nilai berikut :

  • thin ( biasanya 1px )
  • medium ( biasanya 3px )
  • thick ( biasanya 5px )
  • ukuran tertentu ( dalam px, pt, cm, em, dll )
Contoh berikut menunjukkan beberapa outline dengan width yang berbeda :

outline - width Properti

thin outline ( garis tipis )

medium outline ( garis luar sedang )

thick outline ( garis luar yang tebal )

4px thick outline ( garis luar yang tebal 4px )

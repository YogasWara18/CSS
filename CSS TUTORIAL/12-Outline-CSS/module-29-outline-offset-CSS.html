Semua properti outline CSS

Property Description
outline Untuk menambahkan garis di sekitar elemen HTML, diluar border, tanpa mempengaruhi ukuran atau layout elemen tersebut.
outline-color Digunakan untuk menentukan warna garis luar ( outline ) dari sebuah elemen HTML. Garis ini berada di luar border, dan berfungsi untuk menonjolkan elemen tanpa mempengaruhi ukuran atau tata letaknya.
outline-offset Digunakan untuk mengatur jarak antara outline dan tepi elemen. Properti ini memberi efek visual seolah - olah outline " melayang " atau menjauh dari elemen, tanpa mempengaruhi layout ata ukuran elemen itu sendiri.
outline-style Digunakan untuk menentukan bentuk atau gaya garis luar ( outline ) dari sebuah elemen HTML. Outline ini berada diluar border dan berfungsi untuk menonjolkan elemen, terutama saat interaksi seperti fokus atau hover.
outline-width Digunakan untuk menambahkan ketebalan garis luar ( outline ) dari sebuah elemen HTML. Outline ini berada diluar border dan tidak mempengaruhi ukuran atau tata letak elemen.

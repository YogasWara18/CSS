CSS Outline Color

Properti outline-color ini digunakan unutk mengatur warna outline.
Warna dapat diatur dengan :

  • Nama - tentukan nama warna, seperti "red"
  • HEX - tentukan nilai hex, seperti "#ff0000"
  • RGB - tentukan nilai RGB, seperti "rgb(255,0,0)
  • HSL - tentukan nilai HSL, seperti "hsl(0, 100%, 50%)
  • Invert - melakukan inversi warna ( yang memastikan bahwa garis luar terlihat terlepas dari warna latar belakang )

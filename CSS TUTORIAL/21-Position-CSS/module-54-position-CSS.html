CSS Positioning

Penempatan CSS adalah tentang mengendalikan penempatan elemen dalam halaman web.

Dengan pemosisian CSS, Anda dapat mengesampinkan alur dokumen normal.

Property Posisi CSS

Properti position menentukan jenis posisi untu suatu elemen.
Properti ini dapat memiliki salah satu nilai berikut :

  • static - ini adalah nilai default
  • relative
  • fixed
  • absolute
  • sticky

Elemen kemudian diposisikan ke lokasi akhornya dengan properti,,, dan . top bottom left right

Poisisi CSS : Static

Semua elemen HTML diposisikan statis secara default.
Elemen yang diposisikan statis tidak terpengaruh oleh properti atas, bawah, kiri, dan kanan.

Posisi CSS: absolute

Elemen dengan position: absolute; diposisikan relatif terhadap leluhur yang diposisikan terdekat ( dengan posisi selain statis ).
Akan tetapi; jika elemen diposisikan absolut tidak memiliki leluhur yang diposisikan, elemen tersebut menggunakan badan dokumen, dan bergerak seiring dengan pengguliran halaman.
Catatan : Elemen yang diposisikan absolut dihapus dari alur dokumen normal, dan dapat tumpang tindih dengan elemen lainnya.

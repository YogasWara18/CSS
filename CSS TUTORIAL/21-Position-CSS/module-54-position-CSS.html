CSS Positioning

Penempatan CSS adalah tentang mengendalikan penempatan elemen dalam halaman web.

Dengan pemosisian CSS, Anda dapat mengesampinkan alur dokumen normal.

Property Posisi CSS

Properti position menentukan jenis posisi untu suatu elemen.
Properti ini dapat memiliki salah satu nilai berikut :

  • static - ini adalah nilai default
  • relative
  • fixed
  • absolute
  • sticky

Elemen kemudian diposisikan ke lokasi akhornya dengan properti,,, dan . top bottom left right

Poisisi CSS : Static

Semua elemen HTML diposisikan statis secara default.
Elemen yang diposisikan statis tidak terpengaruh oleh properti atas, bawah, kiri, dan kanan.

Contoh :

Posisi CSS : relative

elemen dengan position : relative; diposisikan relatif terhadap posisi normalnya dalam alur dokumen.

Mengatur properti atas, kanan, bawah, dan kiri akan menyebabkan elemen disesuaikan menjauh dari posisi normalnya. konten lainnya tidak akan disesuaikan agar muat di celah yang ditinggalkan oleh elemen.

Posisi CSS : fixed

Elemen dengan position: fixed; diposisikan relatif terhadap viewport, yang berarti elemen tersebut akan selalu berada di tempat yang sama meskipun halaman digulir. Propery atas, kanan, bawah, dan kiri digunakan untuk menentukan lokasi akhir elemen.
Elemen tetap tidak meningggalkan celah pada halaman tempat elemen tersebut biasanya berada

Elemen div ini memiliki property position: fixed

Posisi CSS: absolute

Elemen dengan position: absolute; diposisikan relatif terhadap leluhur yang diposisikan terdekat ( dengan posisi selain statis ).
Akan tetapi; jika elemen diposisikan absolut tidak memiliki leluhur yang diposisikan, elemen tersebut menggunakan badan dokumen, dan bergerak seiring dengan pengguliran halaman.
Catatan : Elemen yang diposisikan absolut dihapus dari alur dokumen normal, dan dapat tumpang tindih dengan elemen lainnya.

Elemen ini memiliki property position: relative;

Posisi CSS: sticky

elemen dengan position: sticky; tombol antara posisi relative dan fixed, tergantung pada posisi gulir.

Elemen sticky diposisikan relatif hingga posisi gulir tertentu tercapai - lalau "menempel" di tempat itu ( seperti posisi: fixed ). Catatan : Anda harus menentukan setidaknya satu properti top, right, bottom atau left, agar pemosisian sticky dapat berfungsi.

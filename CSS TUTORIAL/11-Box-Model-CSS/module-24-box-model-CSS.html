CSS Box Model

Semua elemen HTML dapat dianggap sebagai BOX

Box-model CSS

Dalam CSS, istilah "box model" digunakan ketika berbicara tentang desain dan tata letak.
Box-model CSS pada dasarnya adalah kotak yang membungkus setiap elemen HTML Model ini terdiri dari: konten, padding, border, dan margin. Gambar di bawah ini mengilustrasikan box-model tersebut :

Penjelasan bagian-bagian yang berbeda :

  • Konten - konten box, tempat teks dan gambar muncul
  • Padding - Berfungsi untuk memberi ruang didalam elemen, yaitu jarak antara konten ( seperti teks atau gambar ) dan batas elemen ( border )
  • Border - Batas yang mengelilingi padding dan konten
  • Margin - Membersihkan area diluar batas. Margin transparan

Box model memungkinkan kita untuk menambahkan batas disekitar elemen, dan menentukan ruang antar elemen.

Width & Height Element

Untuk mengatur width dan height elemen dengan benar disemua browser, anda perlu mengetahui cara kerja box model.

Penting : Saat anda mengatur properti width dan heihgt suatu elemen dengan CSS , Anda hanya perlu mengatur width dan height area konten. Untuk menghitung total width dan height suatu elemen, Anda juga harus menyertakan padding dan border.

Contoh :
Elemen < div > ini akan memiliki width 350px dan height 80px:

Menghitung total width :

Gambar diatas memiliki width 350px. width elemen ini juga 350px. Height total elemen ini adalah 80px.

Berikut perhitungannya:

        320px ( lebar area konten )
        + 20px ( padding kiri + padding kanan )
        + 10px ( border kiri + border kanan )
        = 350px ( lebar total )

        50px ( tinggi area konten )
        + 20px ( padding atas + padding bawah )
        + 10px ( batas kiri + batas kanan )
        = 80px ( tinggi total )

Lebar total suatu elemen harus dihitung seperti ini :
lebar elemen total = width + padding kiri + border kiri + border kanan

height total suatu elemen harus dihitung seperti ini :
height elemen total = height + padding atas + padding bawah + border atas = border bawah

Catatan : Properti mrgin juga mempengaruhi total ruang yang akan digunakan box di halaman, tetapi margin tidak termasuk dalam ukuran box yang sebenarnya. Width dan height total box berhenti di tepinya.

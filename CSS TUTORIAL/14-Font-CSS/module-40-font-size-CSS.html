CSS Font Size

Properti CSS font-size digunakan untuk menentukan ukuran teks/font.
Mampu mengelola ukuran teks sangat penting dalam desain web.
Namun, anda sebaiknya tidak menggunakan penyesuaian ukuran font untuk membuat paragraf tampak seperti judul, atau judul tampak seperti paragraf. selalu gunakan tag HTML yang tepat, seperti < h1 > - < h6 > untuk judul dan < p > untuk paragraf.
Dapat font-size diatur ke ukuran absolut atau ukuran relatif.
Ukuran absolut :

  • px : Pixel menawarkan kontrol yang tetap dan tepat atas ukuran font .
  • xx-small,x-small,small,medium,large,x-large,xx-large Kata kunci ini memiliki serangkaian ukuran yang telah ditentukan sebelumnya di browser.
Ukuran relatif :
  • em : Unit ini relatif terhadap font element induk.
  • rem : Unit ini relatif terhadap ukuran font elemen HTML root.
  • % : Unit ini relatif terhadap ukuran font elemen induk.
  • Smaller dan larger : Unit ini menyesuaikan ukuran font relatif terhadap elemen induk.


Bagaimana memilih unit yang tepat?

  • Untuk kontrol yang tepat dan tepat, px dapat digunakan.
  • Dengan px, halaman web tidak akan berskala baik pada berbagai ukuran layar atau dengan preferensi pengguna.
  • Ukuran absolut dapat berguna ketika ukuran fisik keluaran diketahui.
  • Untuk desain web yang dapat diskalakan, gunakan em atau rem ; mereka memungkinkan pengguna untuk menyesuaikan ukuran teks di pengaturan browser mereka.
  • Persentase ( % ) dapat berguna untuk menyesuaikan ukuran font berdasarkan elemen induk.


Atur Ukuran Font Dengan Piksel

Mengatur ukuran teks dengan piksel ( px ) memberi anda kendali penuh atas ukuran teks.

Jika anda menggunakan piksel, halaman web mungkin tidak berskala dengan baik pada berbagai ukuran layar dan pengguna tidak dapat menyesuaikan ukuran teks di pengaturan peramban mereka. Namun, pengguna masih dapat menggunakan alat zoom untuk mengubah ukuran keseluruhan halaman.
Contoh :

Properti font-size menggunakan value px

Heading ini menggunakan properti font-size dengan value menggunakan px ( piksel ) 40px

Heading ini menggunakan properti font-size dengan value menggunakan px ( piksel ) 30px

Paragraf ini menggunakan properti font-size dengan value menggunakan px ( piksel ) 16px

Atur ukuran Font Dengan Em

Satuannya em relatif terhadap ukuran font elemen induk. Jadi, jika elemen induk memiliki ukuran font 16 piksel, maka 2,5 piksel akan menghasilkan 40 piksel.

Pada contoh berikut, ukuran teks dalam em piksel sama dengan contoh sebelumnya. Namun, em satuan ini menmungkinkan pengguna untuk menyesuaikan ukuran teks di pengaturan peramban.
Contoh :

Properti font-size menggunakan value em

Heading ini menggunakan properti font-size dengna value menggunakan em

Heading ini menggunakan properti font-size dengan value menggunakan em

Paragraf ini menggunakan properti font-size dengan value menggunakan em

Atur ukuran Font Dengan Rem

Satuannya rem relatif terhadap ukuran font elemen HTML akar ( < html > )

Berbeda dengan em, yang relatif terhadap ukuran font element induknya, `font rem -size` selalu mengacu pada ukuran font elemen < html >, terlepas dari posisinya di pohon dokumen. Hal ini rem sangat berguna untuk menciptakan desain yang skalabel dan responsif. Dengan mengubah ukuran font elemen < html >, semua elemen yang berukuran rem satuan akan diskalakan secara proporsional di seluruh halaman.

Ukuran font default elemen < html > di sebagian besar peramban adalah 16 piksel. Jadi, secara default, 1rem sama dengan 16 piksel kecuali jika ditimpa secata eksplisit di CSS.
Contoh :

Properti font-size menggunakan value rem

Heading ini menggunakan properti font-size dengan value 2.5rem

Heading ini menggunakan properti font-size dengan value 1.875rem

Paragraf ini menggunakan properti font-size dengan value 1rem

The VW Unit

Ukuran font juga dapat diatur dengan vw unit, yang berarti "lebar viewport".
Satuan tersebut vw merupakan satuan relatif yang menyatakan persentase lebar area pandang.
1vw = 1% dari lebar viewport browser saat ini. Jadi, jika lebar viewport 500 piksel, 1vw adalah 5 piksel.

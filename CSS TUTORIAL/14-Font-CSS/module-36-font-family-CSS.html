Pemilihan font itu penting

Memilih font yang tepat memeiliki dampak besar pada bagaimana pembaca menikmati situs web.

Font yang tepat dapat menciptakan identitas yang kuat untuk merek anda.

Memilih font yang mudah dibaca itu penting. Font menambah nilai pada teks Anda. penting juga untuk memilih warna dan ukuran teks yang tepat untuk font tersebut.

Properti font-family CSS

Properti CSS font-family menentukan font untuk suatu elemen.

Tips : properti ini font-family sebaiknya menampung beberapa nama font sebagai sistem "fallback". Jika peramban tidak mendukung font pertama, peramban akan mencoba font berikutnya. Nama - nama font harus dipisahkan dengan koma.

Selalu mulai dengan font yang anda inginkan, dan selalu diakhiri dengan keluarga generik, untuk membiarkan browser memeilih font yang serupa dalam keluarga generik, jika tidak ada font lain yang tersedia.

Catatan : Jika nama font lebih dari satu, harus diapit tanda kutip, seperti: "Times New Roman".

font-family umum ( Generic )

Dalam CSS ada lima keluarga font generik :

  1. Serif memiliki goresan di tepi setiap huruf. Hal ini menciptakan kesan formalitas dan elegan.
  2. Sans-serif memiliki garis - garis yang bersih ( tanpa goresan kecil ). Font ini menciptakan tampilan modern dan minimalis.
  3. Monospace di sini semua huruf memiliki lebar tetap yang sama. Font ini menciptakan tampilan mekanis.
  4. Cursive meniru tulisan tangan manusia.
  5. Fantasy merupakan font yang bersifat dekoratif dan menyenangkan.
Semua nama font yang berbeda termasuk dalam salah satu font-family generic.

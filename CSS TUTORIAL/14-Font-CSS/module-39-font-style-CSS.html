CSS Font Style

Properti CSS font-style menentukan gaya font untuk teks.

properti ini dapat memiliki salah satu nilai berikut :

  • normal - Teks ditampilkan secara normal
  • italic - Teks ditampilkan dalam huruf miring
  • oblique - Teks "condong" ( miring sangat mirip dengan miring )
Contoh :

Properti font-style

paragraph ini menggunakan properti font-style dengan value normal

paragraph ini menggunakan properti font-style dengan value italic

paragraph ini menggunakan properti font-style dengan value oblique

CSS Font Weight

Properti CSS font-weight menentukan seberapa tebal dan tipis karakter dalam teks harus ditampilkan.
Properti ini dapat memiliki salah satu nilai berikut :

  • Normal - ini default. menentukan karakter normal
  • Bold - mendefinisikan karakter tebal
  • Bolder - mendefinisikan karakter yang lebih tebal
  • Lighter - mendefinisikan karakter yang lebih ringan
  • 100 - 900 - menentukan karakter dari tipis ke tebal. 400 sama dengan normal, dan 700 sama dengan tebal.

CSS Font Variant

Properti CSS font-variant menentukan apakah teks akan ditampilkan dalam huruf kapital kecil atau tidak.

Dalam font huruf kecil, semua huruf kecil diubah menjadi huruf kapital. Namun, huruf kapital yang dikonversi muncul dalam ukuran font yang lebih kecil daripada huruf kapital asli dalam teks.

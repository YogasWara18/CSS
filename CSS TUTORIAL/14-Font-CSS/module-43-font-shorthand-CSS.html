The CSS Font Shorthand Property

Untuk mempersingkat kode, dimungkinkan untuk menentukan semua properti font individual dalam satu deklarasi
Properti CSS font adalah properti singkatan untuk :

  • font-style
  • font-variant
  • font-weight
  • font-stretch
  • font-size
  • line-height
  • font-family

Aturan untuk Properti CSS Shorthand

  1. Nilai font-size dan font-family wajib diisi. Jika salah satu nilai lainnya tidak ada, milai defaultnya akan digunakan.
  2. Jika didefinisikan, dan harus mendahului font-style font-variant font-weight font-size
  3. Jika didefinisikan, harus segera mengikuti, didahului oleh "/", seperti ini: 15px/30px. line-height font-size
  4. font-family harus menjadi nilai terakhir yang ditentukan.

CSS Web Safe Fonts

Apa itu FontS Web Safe?

Font web yang safe adalah font yang dipasang secara universal disemua browser dan perangkat.

Fallback Fonts

Namun, tidak ada font yang 100% aman untuk web. Selalu ada kemungkinan font tidak ditemukan atau tidak terpasang dengan benar

Oleh karena itu, sangat penting untuk selalu menggunakan font fallback.
Ini berarti anda harus menambahkan daftar "font cadangan" yang serupa di font-family properti. Jika font pertama tidak berfungsi, peramban akan mencoba font berikutnya, lalu font berikutnya, dan seterusnya. Selalu akhiri daftar dengan nama keluarga font generik.

Contoh
Disini, terdapat tiga jenis font : Tahoma, Verdana, dan Sans-serif. Font dan ketiga adalah cadangan, jika Font pertama tidak ditemukan.

Properti Fallback Fonts

Font Wec Safe terbaik untuk HTML dan CSS

Berikut ini adalah daftar font web safe terbaik untuk HTML dan CSS :

  • Arial ( tanpa serif )
  • Verdana ( tanpa serif )
  • Tahoma ( tanpa serif )
  • Trebuchet MS ( tanpa serif )
  • Times New Roman ( Serif )
  • Georgia ( Serif )
  • Courier New ( monospace )
  • Brush Script MT ( cursive )
Catatan : Sebelum anda menerbitkan situs web anda, selalu periksa bagaimana font and muncul di berbagai browser dan perangkat , dan selalu gunakan fallback fonts

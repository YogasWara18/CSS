CSS Google Fonts

Jika anda tidak ingin menggunakan font standart apa pun dalam HTML, Anda dapat menggunakan Google Fonts.
Google Fonts gratis untuk digunakan, dan memiliki lebih dari 1000 font untuk dipilih.

Cara menggunakan Google Fonts

Cukup tambahkan tautan lembar gaya khusus di bagian < head > lalu rujuk font di CSS.
Contoh :

Heading ini menggunakan properti font-family dengan value sofia, sans-serif

Paragraf ini menggunakan properti font-family dengan value Sofia, sans-serif

12345678910

Contoh
Di sini, kami ingin menggunakan font bernama "Trirong" dari Google Fonts :

Heading ini menggunakan properti font-family dengan value "Trirong", serif

Paragraf ini menggunakan properti font-family dengan value "Trirong", serif

12345678910

Contoh
Disini, kami ingin menggunakan font bernama "Audiowide" dari Google Fonts :

Heading ini menggunakan properti font-family dengan value "Audiowide", sans-serif

Paragraf ini menggunakan peroperti font-family dengna value "Audiowide", sans-serif

12345678910

CATATAN : Saat menggunakan font di CSS, selalu cantumka minimal satu font fallback ( untuk menhidari perilaku yang tidak terduga ). Jadi, di sini anda juga harus menambahkan keluarga font generik ( seperti seerif atau sans-serif ) di akhir daftar

Gunakan beberapa font google

Untuk menggunakan beberapa font google, pisahkan saja nama font dengan karakter pipa ( | ), seperti ini :
Contoh :

Properti ini menggunakan Multiple Google Fonts

Audiowide Font

Sofia Font

Trirong Font

Catatan : Meminta beberapa font dapat memperlambat halaman web Anda! Jadi, berhati - hatilah.

Menata ( styling ) Google Fonts

Tentu saja anda dapat menata Google Fonts sesuai keinginan, dengan CSS!
Contoh :

heading ini menggunakan poroperti font-family dengan styling dan value Sofia Font

Paragraf ini menggunakan properti font-family dengan styling dan value Sofia font

12345678910

Mengaktifkan efek font

Google juga telah mengaktifkan berbagai efek font yang dapat Anda gunakan.
Pertama, tambahkan ke Google API, lalu tambahkan nama kelas khusus ke elmen yang akan menggunakan efek khusus tersebut. Nama kelas selalu diawali dan diakhiri dengan .effect=effectname font-effect- effectname
Contoh :

Heading ini menggunakan properti font-family dengan value "Sofia", sans-serif & mengaktifkan effectnya

Paragraf ini menggunakan properti font-family dengan value "Sofia", sans-serif & mengaktifkan effectnya

12345678910

Untuk meminta beberapa efek font, pisahkan saja nama efek dengan karakter pipa ( | )
Contoh :

Heading ini menggunakan properti font-family dengan value "Sofia", sans-serif dengan effect Neon

Heading ini menggunakan peoperti font-family dengan value "Sofia", sans-serif dan effect outline

Heading ini menggunakan properti font-family dengan value "Sofia", sans-serif dan effect emboss

Heading ini menggunakan properti font-family dengan value "Sofia", sans-serif dan effect shadow-multiple

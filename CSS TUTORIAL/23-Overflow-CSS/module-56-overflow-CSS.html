CSS Overflow Property

Properti CSS overflow mengontrol apa yang terjadi pada konten yang terlalu besar untuk dimuat dalam suatu area.
Menentukan apakah akan memotong konten atau menambahkan bilah gulir ketika konten suatu elemen terlalu besar.
Properti ini overflow memiliki nilai - nilai berikut :

  • visible - Default. Overflow tidak terpotong. Konten dirender diluar kotak elemen.
  • hidden - Luapan terpotong, dan sisa konten disembunyikan
  • scroll - Bilah gulir telah ditambahkan. Pengguna harus menggulir untuk melihat semua konten.
  • auto - Mirip dengan scroll, tetapi menambahkan bilah gulir hanya bila diperlukan

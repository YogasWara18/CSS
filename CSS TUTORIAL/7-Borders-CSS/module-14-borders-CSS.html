CSS Borders

Properti border css memungkinkan anda menentukan gaya, lebar, dan warna border elemen.

CSS Border Style

Properti border-style menentukan jenis batas yang akan ditampilkan.
Nilai berikut diizinkan :

  • dotted - Menentukan batas putus - putus
  • dashed - Menentukan batas putus - putus
  • solid - Menentukan batas padat
  • double - Menentukan batas ganda
  • groove - Menentukan batas beralur 3D. Efeknya bergantung pada nilai warna batas
  • ridge - Menentukan batas beralur 3D. Efeknya bergantung pada nilai warna batas
  • inset - Menentukan batas sisipan 3D. efeknya bergantung pada nilai warna batas
  • outset - Menentukan batas awal 3D. Efeknya bergantung pada nilai warna batas
  • none - Tidak menentukan batas
  • hidden - Menentukan batas tersembunyi

Properti border-style dapat memiliki satu hingga empat nilai ( untuk batas atas, batas kanan, batas bawah, dan batas kiri ).
Contoh
Demonstrasi berbagai gaya border : 
            p.dotted {border-style: dotted;}
            p.dashed {border-style: dashed;}
            p.solid {border-style: solid;}
            p.double {border-style: double;}
            p.groove {border-style: groove;}
            p.ridge {border-style: ridge;}
            p.inset {border-style: inset;}
            p.outset {border-style: outset;}
            p.none {border-style: none;}
            p.hidden {border-style: hidden;}
            p.mix {border-style: dotted dashed solid double;}

Semua properti Border CSS

Property Description
border : Mengatur semua properti perbatasan dalam satu deklarasi.
border-bottom : Mengatur semua properti border bawah dalam satu deklarasi
border-bottom-color : Mengatur warna border bawah
border-bottom-style : Mengatur style border bawah
border-bottom-width : Mengatur lebar border bawah
border-color : Mengatur warna keempat border
border-left : Mengatur semua properti border kiri dalam satu deklarasi
border-left-color : Mengatur color border kiri
border-left-style : Mengatur style border kiri
border-left-width : Mengatur lebar border kiri
border-radius : Mengatur keempat properti border-radius untuk membuat sudut membulat
border-right : Mengatur semua properti border yang tepat dalam satu deklarasi
border-right-color : Mengatur warna border kanan
border-right-style : Mengatur style border kanan
border-right-width : Mengatur lebar border kanan
border-style : Mengatur semua properti border atas dalam satu deklarasi
border-top-color : Mengatur warna border atas
border-top-style : Mengatur style border atas
border-top-width : Mengatur lebar border atas
border-width : Mengatur lebar keempat border

CSS Border - Individual Sides

Dari contoh dihalaman sebelumnya, Anda telah melihat bahwa dimungkinkan untuk menentukan batas yang berbeda untuk setiap sisi.
Dalam CSS, ada juga properti untuk menentukan setiap batas ( atas, kanan, bawah, dan kiri) :
Contoh :

Individual Border Sides

2 different border styles.

Contoh diatas memeberikan hasil yang sama seperti ini :

menggunakan cara yang lebih singkat

Jika border-style properti memiliki empat nilai :

  • border-style: dotted solid double dashed :
  • top border is dotted
  • right border is solid
  • bottom border is double
  • left border is dashed

jika border-style properti memiliki tiga nilai :

  • border-style: dotted solid double :
  • top border is dotted
  • right and left borders are solid
  • bottom border is double

jika border-style properti memiliki dua nilai :

  • border-style: dotted solid :
  • top and bottom borders are dotted
  • right and left borders are solid

jika border-style properti memiliki satu nilai :

  • border-style: dotted
  • Keempat batasnya dotted

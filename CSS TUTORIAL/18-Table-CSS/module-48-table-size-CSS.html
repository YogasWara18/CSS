CSS Table Width and Height

Lebar dan tinggi tabel ditentukan oleh properti width dan height

CSS Table Width ( lebar tabel CSS )

Properti CSS width digunakan untuk mengatur lebar tabel.
lebarnya dapat diatur :

  • Dalam persen ( % )
  • Sebagai panjang tetap ( px )
  • Dengan auto kata kunci

CSS Table Width in Percent ( Lebar Tabel CSS dalam Persen )

Untuk membuat tabel yang membentang di seluruh layar ( lebar penuh ), gunakan width: 100%;

Contoh table layar penuh menggunakan 100 percent ( % )

Untuk membuat tabel yang mencakup setengah halaman, gunakan width: 50%;

Contoh table menggunakan value 50%

Firstname Lastname Savings

Lois Grifon $200
Joe Swanson $300
Cleveland Coklat $260

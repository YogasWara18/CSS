CSS Table Width ( lebar tabel CSS )

Properti CSS width digunakan untuk mengatur lebar tabel.
lebarnya dapat diatur :

  • Dalam persen ( % )
  • Sebagai panjang tetap ( px )
  • Dengan auto kata kunci

CSS Table Width in Percent ( Lebar Tabel CSS dalam Persen )

Untuk membuat tabel yang membentang di seluruh layar ( lebar penuh ), gunakan width: 100%;

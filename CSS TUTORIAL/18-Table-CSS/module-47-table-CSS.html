CSS Table Borders ( Batas Tabel CSS )

Properti CSS digunakan untuk menentukan lebar, gaya, dan warna batas tabel. border
Properti ini merupakan singkatan dari : border

  • border-width - mengatur lebar perbatasan
  • border-style - mengatur gaya perbatasan ( diperlukan )
  • border-color - mengatur warna perbatasan

Mengapa Batas Ganda?

Perhatikan bahwa tabel pada contoh di atas memiliki batas ganda. Hal ini disebabkan oleh elemen < table >, < th >, dan < td > yang memiliki batas terpisah .

CSS Collapse Table Borders ( Batas Tabel ciutkan CSS )

Properti CSS border-collapse menentukan apakah batas tabel akan diciutkan menjadi satu batas tunggal atau dipisahkan seperti pada HTML standar.
Properti ini dapat memiliki salah satu nilai berikut :

  • Separate - Nilai default. Batas dipisahkan; setiap sel akan menampilkan batasnya sendiri
  • Collapse - Batas - batas dilipat menjadi satu batas jika memungkinkan

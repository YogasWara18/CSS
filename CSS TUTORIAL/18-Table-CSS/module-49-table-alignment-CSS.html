CSS Table Alignment

Horizontal Alignment

Properti CSS text-align digunakan untuk mengatur perataan horizontal konten dalam < th > atau < td >.

Properti ini dapat memiliki salah satu nilai berikut :

  • left - Menyelaraskan teks ke kiri
  • right - Menyelaraskan teks ke kanan
  • center - Memusatkan teks
Catatan : Secara default, konten elemen < th > rata tengah dan konten elemen < td > rata kiri!

CSS text-align: center

Untuk Menyelaraskan konten elemen < td > ke tengah, gunakan text-align: center :

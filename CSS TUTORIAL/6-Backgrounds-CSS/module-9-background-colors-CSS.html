CSS BACKGROUNDS

Properti background CSS digunakan untuk menambahkan efek latar belakang untuk elemen.

Dalam bab ini, akan mempelajari tentang properti latar belakang CSS berikut :

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
  • background(properti singkatan)

CSS background-color

properti bcakground-color menentukan warna latar belakang suatu elemen.

Contoh
Warna latar belakang halaman diatur seperti ini : 

        body {
            background-color: lightblue;
        }

Dengan CSS, warna paling sering ditentukan oleh :

  • Nama warna yang valid - seperti "red"
  • Nilai HEX - seperti "#ff0000"
  • Nilai rgb(255, 0, 0)

Other Element ( Elemen Lainnya )

Anda dapat mengatur warna latar belakang untuk elemen HTML apa pun :

Contoh
Disini, elemen < h1 >, < p >, dan < div > akan memilih warna latar belakang yang berbeda : 

        h1 {
            background-color: green;
        }

        div {
            background-color: lightblue;
        }

        p {
            background-color: yellow;
        }

Opacity / Transparansi

Properti ini opacity menentukan opasitas/transparansi suatu elemen. Nilainya dapat berkisar dari 0.0 hingga 1.0 semakin rendah nilainya, semakin transparan :

Contoh 

        div {
            background-color: green;
            opacity: 0.3;
        }

Catatan :Saat menggunakan opacity properti untuk menambahkan transparansi pada latar belakang suatu elemen, semua elemen turunannya mewarisi transparansi yang sama. hal ini dapat membuat teks didalam elemen yang sepenuhnya transparan sulit dibaca.

Transparansi menggunakan RGBA

Jika anda tidak ingin menerapkan opasitas pada elemen anak, seperti pada contoh di atas, gunakan nilai warna RGBA. Contoh berikut menetapkan opasitas untuk watna latar belakang, bukan teks.
Anda telah mempelajari dari Bab warna CSS kami bahwa Anda dapat menggunakan RGB sebagai nilai warna. Selain RGB, Anda dapat menggunakan nilai warna RGB dengan kanal alfa ( Rgb A ) - yang menentukan opasitas suatu warna.
Nilai warna RGBA ditentukan dengan: rgba(red, green, blue, alfa). parameter alfa berupa angka antara 0.0 (sepenuhnya transparan) dan 1.0 (sepenuhnya buram).
Tip : Anda akan mempelajari lebih lanjut tentang Warna RGBA di Bab warna CSS :
Contoh 

        div {
            background: rgba(0, 128, 0, 0.3) /* green background with 30% opacity */
        }

CSS Background Attachment ( Lampiran latar belakang CSS )

Background attachment CSS

Properti background-attachment menentukan apakah gambar background harus digulir atau tetap ( tidak akan digulir bersama bagian halaman lainnya ) :
Contoh :
Tentukan bahwa background harus diperbaiki : 

        body {
            background-image: url("img_tree.png");
            background-repeat: no-repeat;
            background-position: right top;
            background-attachment: fixed;
        }
Contoh :
Tentukan bahwa background harus bergulir bersama dengan sisa halaman : 
        body {
            background-image: url("img_tree.png");
            background-repeat: no-repeat;
            background-position: right top;
            background-attachment: scroll;
        }

Properti background-attachment CSS

Property Description
background-attachment mengatur apakah background image fixed ( tetap ) atau scroll ( bergulir ) bersama bagian halaman lainnya

List style CSS ( Daftar Gaya CSS )

Dalam HTML, ada dua jenis utama list daftar :

  • < ul > - daftar list tidak berurutan ( item daftar ditandai dengan poin hitam )
  • < ol > - daftar berurutan ( item dafatar ditandai dengan angka atau huruf )


CSS memiliki properti berikut untuk menata daftar HTML :
  • list-style-type - Menentukan jenis penanda item daftar
  • list-style-image - Menentukan gambar sebagai penanda item daftar
  • list-style-position - Menentukan posisi penanda item daftar
  • list-style - Properti singkatan untuk properti di atas

CSS Hapus Penanda Item Daftar

Properti list-style-type: none; ini digunakan untuk menghapus penanda item daftar.

Catatan : Daftar memiliki margin dan padding default. Untuk menghapusnya, tambahkan margin:0 dan padding:0 ke elemen < ul > atau < ol > :

CSS list-style Shorthand property ( Properti singkatan Gaya Daftar CSS )

Properti ini list-style adalah properti singkat ( shorthand ). Properti ini digunakan untuk mengatur semua properti daftar dalam satu deklarasi.

Saat menggunakan properti singkatan ( shorthand ) urutan nilai properti adalah :

  • list-style-type
  • list-style-position
  • list-style-image

Jika salah satu nilai properti diatas hilang, nilai default untuk properti yang hilang akan dimasukkan.
Contoh :

the list-style property shorthand

  • Coffee
  • Tea
  • Coca - cola

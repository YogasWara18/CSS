Pengaturan CSS Height dan Width

Properti height dan width digunakan untuk mengatur tinggi dan lebar suatu elemen.
Properti height dan width tidak mencakup padding, border, atau margin. Properti ini mengatur height/width area didalam padding, border, dan margin elemen.

Value height dan width CSS

Properti height dan width dapat memiliki nilai berikut :

  • auto - Ini default. Peramban menghitung height dan width
  • length - Menentukan height/width< dalam px, cm, dll.
  • % - Menentukan height/width dalam persen dari blok yang berisi
  • initial - Mengatur height/width ke nilai defaultnya
  • inherit - Height/width akan diwarisi dari nilai induknya

Mengatur Max-Width ( Lebar maksimum )

Properti max-width ini digunakan untuk mengatur width maksimum suatu elemen.
Dapat max-width ditentukan dalam nilai length ( panjang ), seperti px, cm, dsb., atau dalam persen ( %) dari blok yang memuatnya, atau ditetapkan ke tidak ada ( ini adalah default. Berarti tidak ada lembar maksimum ).

Catatan : Jika karena alasan tertentu anda menggunakan kedua width properti tersebut dan max-width properti pada elemen yang sama, dan nilai properti width lebih besar dari max-width properti tersebut; max-width properti tersebut akan digunakan ( dan width properti tersebut akan diabaikan ).

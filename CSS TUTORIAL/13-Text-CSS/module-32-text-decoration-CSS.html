CSS Text Decoration ( Dekorasi Teks )

Dalam bab ini anda akan mempelajari tentang properti berikut:

  • text-decoration-line
  • text-decoration-color
  • text-decoration-style
  • text-decoration-thickness
  • text-decoration

Tambahkan Garis Dekorasi ke Teks :

Properti ini text-decoration-line digunakan untuk menambahkan garis dekorasi pada teks.
Kiat : Anda dapat menggabungkan lebih dari satu nilai, seperti garis atas dan garis bawah untuk menampilkan garis di atas dan dibawah teks.
Contoh :

Property text-decoration-line :

Dekorasi menggunakan value text-decoration-line: overline ( garis atas )

Dekorasi menggunakan value text-decoration-line: line-through ( garis melalui teks )

Dekorasi menggunakna value text-decoration-line: underline ( garis bawah )

Dekorasi menggunakan value text-decoration-line: overline underline ( garis atas dan garis bawah )

Catatan : Tidak disarankan untuk menggarisbawahi teks yang bukan tautan. Karena hal ini sering membingungkan pembaca.

Menambahkan warna untuk text-decoration :

Properti text-decoration-color ini digunakan untuk mengatur warna dan garis dekorasi.

Text decoration menggunakan properti text-decoration-line & text-decoration-color value : overline & red ( Garis atas dengan warna merah )

Text decoration menggunakan properti text-decoration-line & text-decoration-color value line-through & blue ( Garis melalui teks dengan warna biru )

Text decoration menggunakan properti text-decoration-line & text-decoration-color value underline & green ( Garis dibawah teks dengan warna hijau )

Text decoration menggunakan properti text-decoration-line & text-decoration-color value overline underline & purple ( Garis atas dan bawah diantara teks dengan warna ungu )

Menambahkan style ( gaya ) pada text-decoration :

Properti ini text-decoration-style digunakan untuk mengatur style text decoration.

Text decoration menggunakan properti text-decoration-line & text-decoration-style dengan value overline & solid ( Garis diatas teks dengan gaya tebal )

Text decoration menggunakan properti text-decoration-line & text-decoration-style dengan value underline & double ( Garis dibawah text dengan bergaya double )

Text decoration menggunakan properti text-decoration-line & text-decoration-style dengan value underline & dotted ( Garis dibawah teks dengan gaya titik putus - putus )

Text decoration menggunakan properti text-decoration-line & text-decoration-style dengan value underline & dashed ( Garis dibawah text dengan gaya garis putus - putus )

Teks decoration menggunakan properti text-decoration-line & text-decoration-style dengan value underline & wavy ( Garis dibawah teks dengan gaya bergelombang ( wavy ) )

Text decoration menggunakan properti text-decoration-line & text-decoration-style & text-decoration-color dengan value underline & red & wavy ( Garis dibawah teks dengan warna merah dan gaya bergelombang ( wavy ) )

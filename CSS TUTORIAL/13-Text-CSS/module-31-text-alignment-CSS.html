Text Alignment ( Penyelarasan Teks )

Properti text-align ini digunakan untuk mengatur perataan horizontal suatu teks.
Suatu teks dapat diratakan kiri atau kanan, dipusatkan, atau diberi tepi kanan atau kiri.
Contoh berikut menunjukkan teks yang diratakan tengah, dan diratakan kiri dan kanan ( perataan kiri adalah default jika arah teks dari kiri dan kanan ), dan perataan kanan adalah default jika arah teks dari kanan dan kiri

Ketiga judul diatas disejajarkan ditnagah ( center ), kiri ( left ), right ( kanan )

Ketika text-align properti diatur ke "justify", setiap baris akan diregangkan sehingga setiap baris memiliki lebar yang sama, dan margin kiri dan kanan lurus ( seperti di majalah dan surat kabar ):

text-align: justify

Nilai text-align: justify; Meregangkan garis sehingga setiap garis memiliki lebar yang sama ( seperti di surat kabar dan majalah ).

Text Align Last ( Ratakan Teks Terakhir )

properti text-align-last menentukan cara menyelaraskan baris terakhir teks.
Contoh :

Text-align-last Property ( Perataan Teks Terakhir )

text-align-last: right

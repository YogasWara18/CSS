CSS Text

Css memiliki banyak properti untuk memformat teks.

Text fotmatting

Text ini ditata dengan beberapa properti pemformatan teks. Judulnya menggunakan properti perataan teks, transformasi teks, dan warna.
Paragraf diindentasi, diratakan, dan spasi antar karakter ditentukan. Garis bawah dihapus dari paragraf berwarna ini. "Coba Sendiri" tautan.

Text Color ( Warna teks )

Properti color ini digunakan untuk mengatur warna teks. Warna ditentukan oleh :

  • color name : Seperti " red "
  • HEX value : seperti #ff0000
  • RGB value : rgb(255, 0, 0)
Lihat nilai warna CSS untuk daftar lengkap kemungkinan nilai warna.
Warna teks default untuk halaman ditentukan dalam pemilih badan.

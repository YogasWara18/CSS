Styling Links Depending on State

Selain itu, links dapat diberi gaya berbeda, bergantung pada Negara tempat link itu berada.

Empat status links tersebut adalah :

  • :link - link normal yang belum dikunjungi
  • :visited - link yang telah dikunjungi pengguna
  • :hover - link saat pengguna mengarahkan kursor ke atasnya
  • :active link saat diklik

Saat mengatur gaya untuk status link, ada beberapa aturan urutan :
  • :hover harus datang setelah :link dan :visited
  • :active harus datang setelah :hover

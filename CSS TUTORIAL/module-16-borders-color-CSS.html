CSS Border Color

Properti border-color ini digunakan untuk mengatur warna keempat batas.
Warna dapat diatur dengan :

  • nama - tentukan nama warna, seperti "red"
  • HEX - tentukan nilai HEX, seperti "#ff0000"
  • RGB - tentukan nilai RGB, seperti "rgb(255,0,0)"
  • HSL - tentukan nilai HSL, seperti "hsl(0, 100%, 50%)"
  • Trasparan
Catatan : jika border-color tidak disetel, ia mewarisi warna elemen.

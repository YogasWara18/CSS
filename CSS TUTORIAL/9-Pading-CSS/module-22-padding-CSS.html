CSS Padding

Properti CSS padding digunakan untuk menghasilkan spasi di sekitar konten elemen, di dalam bats yang ditentukan.
Dengan CSS, anda memiliki kendali penuh atas padding. Terdapat properti untuk mengatur padding untuk setiap sisi elemen ( atas, kanan, bawah, dan kiri ).

Padding - Individual Sides

CSS memiliki properti untuk menentukan padding untuk setiap sisi elemen :

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
Semua properti padding dapat memiliki nilai berikut :
  • Panjang - menentukan padding dalam px, pt, cm, dll
  • % - menentukan padding dalam % dari lebar elemen yang memuatnya
  • Mewarisi - menentukan bahwa padding harus diwarisi dari elemen induk
Catatan : Nilai negatif tidak diizinkan.

Padding - Shorthand Property

Untuk mempersingkat kode, dimungkinkan untuk menentukan semua properti padding dalam satu properti.
Properti ini padding adalah properti singkatan untuk properti padding individual berikut :

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Jadi, beginilah cara kerjanya :
jika padding properti memiliki empat nilai :
  • Padding : 25px 50px 75px 100px
  • padding atas adalah 25px
  • padding kanan adalah 50px
  • padding bawah adalah 75px
  • padding kiri adalah 100px

Jika padding properti memiliki tiga nilai :
  • Padding: 25px 50px 75px
  • padding atas adalah 25px
  • padding kanan adalah 50px
  • padding bawah adalah 75px

jika padding properti memiliki dua nilai :
  • Padding: 25px 50px
  • padding atas adalah 25px
  • padding kanan adalah 50px

jika padding properti memiliki satu nilai :
  • Padding: 25px
  • Keempat sudut padding berukuran 25px

Padding and Element Width

Properti CSS width menentukan lebar area konten suatu elemen. Area konten adalah bagian didalam padding, border, dan margin suatu elemen ( model kotak ).

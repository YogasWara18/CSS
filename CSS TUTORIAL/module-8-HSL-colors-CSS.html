CSS HSL COLORS

HSL merupakan singkatan dari HUE ( rona ), SATURATION ( saturasi ), dan LIGHTNESS ( cahaya ).

HSL VALUE ( Nilai HSL )

Dalam CSS, warna dapat ditentukan menggunakan rona, saturasi, dan kecerahan ( HSL ) dalam bentuk :
hsl( rona, saturasi, kecerahan )
Rona ( Hue ) adalah derajat pada roda warna dari 0 hingga 360. 0 adalah merah, 120 adalah hijau, dan 240 adalah biru.
Saturasi ( Saturasion ) adalah nilai persentase. 0% berarti warna abu - abu, dan 100% adalah warna penuh.
Kecerahan ( lightness ) juga merupkan persentase. 0% adalah hitam, 50% tidak terang dan tidak gelap, 100% adalah putih.

Saturation

Saturasi dapat dijelaskan sebagai intensitas warna.
100% adalah warna murni, tidak ada nuansa abu - abu.
50% adalah 50% abu - abu, tetapi anda masih dapat melihat warnanya.
0% sepenuhnya abu - abu; Anda tidak dapat lagi melihat warnanya.

HSL Saturation

hsl(0, 100%, 50%)

hsl(0, 80%, 50%)

hsl(0, 60%, 50%)

hsl(0, 40%, 50%)

hsl(0, 20%, 50%)

hsl(0, 0%, 50%)

Lightness

tingkat kecerahan suatu warna dapat dijelaskan sebagai seberapa banyak cahaya yang ingin anda berikan pada warna tersebut, dimana 0% berarti tidak ada cahaya ( hitam ), 50% berarti 50% cahaya ( tidak gelap maupun terang ) dan 100% berarti kecerahan penuh ( putih ).

Shades of gray ( Nuanda abu - abu )

Nuansa abu - abu sering kali ditentukan dengan mengatur rona (hue) dan saturasi (saturasion) ke 0, dan menyesuaikan kecerahan dari 0% hingga 100% untuk mendapatkan nuansa yang lebih gelap/lebih terang:

Shades of gray

hsl(0, 0%, 0%)

hsl(0, 0%, 24%)

hsl(0, 0%, 47%)

hsl(0, 0%, 71%)

hsl(0, 0%, 94%)

hsl(0, 0%, 100%)

HSLA VALUE ( Nilai HSLA )

Nilai warna HSLA merupakan perluasan nilai warna HSL dengan saluran alfa - yang menentukan opasitas suatu warna.
Nilai warna HSLA ditentukan dengan : hsla( Hue, Saturasi, Lightness, alfa )
Parameter alpha adalah angka antara 0.0 ( sepenuhnya transparan ) dan 1.0 ( tidak transparan sama sekali ):
Lakukan percobaan dengan mencampur nilai HSLA dibawah ini :

Make transparent colors with HSLA

hsla(9, 100%, 64%, 0.2)

hsla(9, 100%, 64%, 0.6)

hsla(9, 100%, 64%, 0.8)

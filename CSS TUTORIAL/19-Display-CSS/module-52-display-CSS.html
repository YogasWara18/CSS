CSS THE DISPLAY PROPERTY

The CSS Display Property

Property ini display merupakan properti CSS penting untuk mengontrol tata letak. Properti ini menentukan apakah suatu elemen HTML diperlakukan sebagai blok atau elemen sebaris.

Setiap elemen HTML memiliki nilai tampilan default, tergantung jenis elemennya. Nilai tampilan default untuk sebagian besar elemen adalah block atau inline.

Properti display ini digunakan untuk mengubah prilaku tampilan default elemen HTML.

Block-level Elements

elemen tingkat blok selalu dimulai pada baris baru dan menempati lebar penuh yang tersedia ( membentang ke kiri atau ke kanan sejauh mungkin ).

Contoh elemen tingkat blok :

  • < div >
  • < h1 > - < h6 >
  • < p >
  • < form >
  • < header >
  • < footer >
  • < section >

Inline elements

Elemen sebaris Tidak dimulai pada baris baru hanya mengambil lebar seperlunya ini adalah elemen < span > sebaris di dalam paragraf.

Contoh elemen inline ( sebaris ) :

  • < span >
  • < a >
  • < img >

Common display values ( nilai tampilan umum )

Properti CSS display memiliki banyak nilai. Tabel berikut mencantumkan nilai - nilai yang paling umum digunakan :

Value Description
inline Menampilkan elemen sebagai elemen sebaris
block Menampilkan elemen sebagai elemen block
contents Membuat wadah menghilang menjadikan elemen anaknya sebagai anak elemen pada level berikutnya di DOM
flex Menampilkan elemen sebagai wadah fleksibel tingkat block
grid Menampilkan elemen sebagai wadah grid tingkat block
inline - block Menampilkan elemen sebagai wadah blok tingkat inline. Elemen itu sendiri diformat sebagai elemen sebaris, tetapi anda dapat menerapkan nilai tinggi, lebar, padding, dan margin.
none Elemen sepenuhnya tersembunyi dari alur dokumen ( tidak memakan tampat apa pun ).

CSS Display : none ;

Saat menggunakan display: none; elemen tersebut sepenuhnya tersembunyi dari aliran dokumen dan tidak memakan tempat apa pun.

Umumnya digunakan dengan javaScript untuk menyembunyikan atau menampilkan elemen tanpa menghapus dan membuatnya kembali.
Contoh :
Cara menggunakan CSS dan JavaScript untuk menampilkan elemen tersembunyi pada acara klik :

Click to show panel

This panel contains a div element, which is hidden by default ( display: none )

kami menggunakan JavaScript untuk menampilkannya ( display: block ).

Cara kerjanya: saat pengguna mengklik elemen p, fungsi bernama myFunction() dijalankan, yang mengubah gaya div dengan id=panel dari display:none ( tersembunyi ) menjadi display: block ( terlihat ).

Anda akan mempelajari lebih lanjut tentang JavaScript di tutorial JavaScript.

CSS HEX COLORS

Warna heksadesimal ditentukan dengan: #RRGGBB, dimana bilangan bulat heksadesimal RR (merah), GG(hijau), dan BB(biru) menentukan komponen warna.

HEX VALUE ( Nilai HEX )

Dalam CSS, warna dapat ditentukan menggunakan nilai heksadesimal dalam bentuk :
#rrggbb
Dimana rr(merah), gg(hijau), dan bb(biru) adalah nilai heksadesimal antara 00 dan ff ( sama dengan desimal 0-255 ).
Misalnya, #ff0000 ditampilkan sebagai merah, karena merah ditetapkan ke nilai tertingginya ( ff ) dan yang lainnya ditetapkan ke nilai terendah ( 00 ).
Untuk menampilkan warna hitam, atur semua nilai ke 00, seperti ini #000000.
Untuk menampilkan warna putih, atur semua nilai ke ff, seperti ini #ffffff.

NILAI HEX 3 Digit

Terkadang anda akan melihat kode HEX 3 digit di sumber CSS.
Kode heksadesimal 3 digit merupakan singkatan dari beberapa kode heksadesimal 6 digit.
kode heksadesimal 3 digit memiliki bentuk sebagai berikut :
#rgb
Dimana r,g,dan b mewakili komponen merah, hijau, dan biru dengan nilai antara 0 dan f.
Kode heksadesimal 3 digit hanya dapat digunakan jika kedua nilai ( RR, GG, dan BB ) sama untuk setiap komponen. Jadi, jika kita memiliki #ff00cc, kodenya dapat ditulis sperti ini #f0c. 

        body {
            background-color: #fc9; /* same as #ffcc99 */
        }

        h1 {
            color: #f0f; /* same as #ff00ff */
        }

        p {
            color: #b58; /* same as #bb5588 */
        }

The CSS z-index property

Properti ini z-index menentukan urutan tumoukan elemen yang diposisikan.
Urutan tumpukan menentukan elemen mana yang harus ditempatkan didepan atau dibelakang elemen lain.
Bila elemen diposisikan, elemen tersebut dapat tumpang tindih dengan elemen lainnya.
Suatu elemen dapat memiliki tatanan tumpukan positif atau negatif ( indeks - z ):
Contoh :

Ini adalah heading menggunakan z-indeks

Karena gambar memiliki z-index -1 gambar akan ditempatkan dibelakang teks.

Catatan : hanya berfungsi pada elemen yang diposisikan ( position : absolute, position : relative, position : fixed, atau position : sticky ) dan item fleksibel ( elemen yang merupakan anak langsung dari elemen display : flex )

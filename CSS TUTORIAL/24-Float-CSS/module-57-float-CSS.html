CSS Float

Properti float menentkan bagaimana suatu elemen harus mengapung dalam wadahnya.
Ia menempatkan elemen disisi kiri atau kanan wadahnya, yang memungkinkan teks dan elemen sebaris untuk membungkusnya.
Gambar di dalam < div > ini melayang ke kanan.
Jika anda memperkecil layar, Anda akan melihat teks melingkari gambar.

Properti CSS Float

Properti ini float digunakan untuk memposisikan dan memformat konten, misalnya membiarkan gambar mengambang ke kiri terhadap teks dalam suatu wadah
Properti ini dapat memiliki salah satu nilai berikut :

  • left - Elemen float disebelah kiri wadahnya
  • right - Elemen mengapung disebelah kanan wadahnya
  • none - Default. Elemen tidak mengambang dan ditampilkan tepat di tempat kemunculannya dalam teks
  • inherit - Elemen mewarisi nilai float dari parent ( induknya )

Tip : Properti ini sering digunakan untuk membungkus teks di sekitar gambar! float

Contoh CSS float: right

Nilai tersebut float: right menunjukkan bahwa suatu elemen harus float ke kanan dalam wadahnya

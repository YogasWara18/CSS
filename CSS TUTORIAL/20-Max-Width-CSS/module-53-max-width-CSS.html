The CSS max - width property

Properti ini max - width menentukan lebar maksimum suatu elemen.
Properti ini mencegah lebar elemen menjadi lebih besar dari nilai yang ditentukan ( bisa lebih kecil, tetapi tidak lebih besar ).
Properti ini berguna untuk membuat tata letak responsif dan memastkan keterbacaan konten di berbagai ukuran layar.

Masalah dengan width :

Disini kita memiliki elemene < div > yang berpusat horizontal dengan spesifik width ( 600px ) :
Apa yang terjadi pada < div > di atas jika jendela peramban lebih kecil dari lebar elemen?
Sebagian konten tidak akan ditampilkan, dan peramban mungkin menambahkan bilah gulir horizontal ke halaman. >

Menggunakan max - width sebagai gantinya :

Sekarang, kita gunakan properti ini sebagai gantinya. Ini akan meningkatkan penanganan jendela kecil oleh peramban : max-width
Tip : Ubah ukuran jendela browser menjadi kurang dari 600px lebarnya, untuk meilhat perbedaan antara kedua < div >
Contoh :

CSS Max-width

This div element has width: 500px;

This div element has max-width: 500px;

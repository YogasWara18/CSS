CSS Icons

Icons dapat dengan mudah ditambahkan ke halaman HTML anda, dengan menggunakan pustaka ikon.

Cara menambahkan Icons

Cara paling sederhana untuk menambakan icons ke halaman HTML Anda adalah dengan pustaka Icons, seperti Font Awesome.
Tambahkan nama kelas ikon yang ditentukan ke elemen HTML sebaris mana pun ( seperti < i > atau < span > ).
Semua icons di pustaka icons dibawah ini , adalah vektor berskala yang dapat disesuaikan dengna CSS ( ukuran, warna, bayangaan, dll. )
